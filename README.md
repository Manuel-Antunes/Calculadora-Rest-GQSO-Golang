# Calculadora-Rest-GQSO-Golang

Uma calculadora que usa requisições HTTP em um servidor de estado representativo para retornar o resultado de operações

## Techs 🚀
* Echo 
* Golang
* Gin/CLI

## Como funciona ? 🧐

Rota de soma
> base_url/sum/:val1/:val2

Rota de subtração
> base_url/sum/:val1/:val2
